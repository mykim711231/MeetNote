# MeetNote 사용설명서

MeetNote는 **내 기기에서만 동작하는 무료 회의록 앱**입니다. 녹음하면 자막이 만들어지고, 요약·할 일까지 자동으로 정리됩니다. 인터넷·계정·요금이 필요 없습니다.

> 가장 잘 동작하는 브라우저: **Chrome** 또는 **Edge** (실시간 자막 때문)

---

## 1. 설치하기 (선택)

설치하면 앱처럼 홈 화면/바탕화면에서 바로 열 수 있고, 오프라인에서도 동작합니다.

1. **설정** 탭으로 이동
2. **“홈 화면에 앱 설치”** 버튼 클릭
3. 안내에 따라 설치 (기기별로 방법이 다르며 화면이 안내합니다)
   - **안드로이드/PC**: 자동 설치 창 또는 주소창의 설치 아이콘
   - **아이폰(Safari)**: 공유 버튼 → “홈 화면에 추가”
   - **삼성 인터넷**: 메뉴 → “페이지를 홈 화면에 추가”

> 녹음과 설치는 보안 연결(https) 또는 localhost에서만 됩니다.

---

## 2. 녹음하기

1. **녹음** 탭으로 이동
2. (선택) **회의 제목**을 입력
3. **자막 언어**를 고릅니다(오른쪽 위, 기본 한국어)
4. **소리 종류**를 고릅니다
   - **마이크**: 직접 말하는 대면 회의
   - **시스템 소리**: 화면 속 온라인 회의(줌·미트·팀즈)의 소리 — 시작 시 공유 창에서 **“탭/시스템 오디오 공유”를 켜세요**
   - **둘 다**: 내 마이크 + 시스템 소리 함께
5. **발언자**를 선택(‘나’/‘상대’)하거나 **＋** 로 추가 — 말하기 전에 누가 말하는지 눌러두면 자막에 표시됩니다
6. 가운데 **빨간 마이크 버튼**을 눌러 시작
7. 말하면 화면 아래에 **자막이 실시간으로** 쌓입니다
8. 버튼
   - **⏸ 일시정지 / ▶ 재개**
   - **✕ 취소**(버림)
   - **■ 저장** — 회의록으로 저장되고 상세 화면이 열립니다

> 💡 녹음 중에는 화면이 자동으로 꺼지지 않습니다.
> 💡 녹음 도중 앱이 꺼져도, 다시 열면 **“중단된 녹음 복구”** 를 제안합니다.

---

## 3. 회의록 보기 · 편집

**기록** 탭에서 회의록을 누르면 상세 화면이 열립니다.

### 재생
- **▶/⏸** 재생/정지, 바를 끌어 위치 이동, **속도** 버튼(1×→1.25×→1.5×→2×→0.75×)
- **자막을 누르면** 그 구간으로 점프합니다
- 키보드: **Space**(재생/정지), **← →**(5초 이동)
- 중간에 닫아도 다음에 열면 **그 지점부터 이어 재생**됩니다

### 4개 탭
- **전문**: 전체 자막. **편집** 버튼으로 내용 수정·발언자 변경·발언 삭제
- **요약**: 핵심 문장 자동 추출
- **할 일**: “~하기로 했어요” 같은 실행 항목 자동 추출
- **메모**: 자유롭게 적는 메모(자동 저장)

### 그 외
- 상단 **📌 핀**: 목록 맨 위로 고정
- **폴더 선택**: 분류
- 제목을 눌러 **수정**
- **🗑 삭제**: 삭제 후 6초 안에 **“실행 취소”** 가능

---

## 4. 찾기 · 정리

**기록** 탭에서:
- 위쪽 **검색창**에 제목이나 내용을 입력 → 일치 부분이 강조됩니다
- **정렬**: 최신순 / 오래된순 / 긴 길이순
- **폴더**로 필터(설정에서 폴더를 먼저 만드세요)
- 상단 **통계**: 총 회의 수, 총 녹음 시간, 이번 주 건수

---

## 5. 내보내기 · 공유

상세 화면 맨 아래 **내보내기** 줄에서:
- **TXT** — 단순 텍스트
- **MD** — 마크다운(요약·할 일·메모 포함)
- **JSON** — 데이터 원본
- **오디오** — 녹음 파일
- **복사** — 마크다운을 클립보드로
- **공유** — 휴대폰 공유 시트(지원 기기)
- **인쇄** — 인쇄 창에서 **“PDF로 저장”** 가능

설정 탭에서 **모든 회의록 합본(.md)** 한 파일로도 내보낼 수 있습니다.

---

## 6. 백업 · 복원 (중요!)

데이터는 **이 기기에만** 저장됩니다. 브라우저 정리나 기기 변경 시 사라질 수 있으니 **정기 백업**을 권장합니다.

**설정 → 백업·복원**
- **전체 백업**: 오디오 포함 (용량 큼)
- **텍스트만 백업**: 가볍게 자막·메모만
- **복원**: 백업 파일(.json)을 불러와 합치기

추가로 **설정 → 저장소 → 영속 저장 활성화** 를 켜두면 브라우저가 공간 부족 시 데이터를 함부로 지우지 않습니다.

---

## 7. 자주 묻는 질문

**Q. 자막이 안 만들어져요.**
A. Chrome/Edge에서 가장 잘 됩니다. 시스템 소리만 녹음하면 자막은 생성되지 않습니다(녹음·재생은 됩니다).

**Q. 마이크가 안 켜져요.**
A. 브라우저의 마이크 권한을 허용했는지 확인하세요. 보안 연결(https)에서만 동작합니다.

**Q. 온라인 회의(줌 등) 소리를 녹음하려면?**
A. 소리 종류를 **시스템 소리** 또는 **둘 다**로 고르고, 시작 시 공유 창에서 **오디오 공유**를 켜세요.

**Q. 데이터가 외부로 전송되나요?**
A. 아니요. 모든 데이터는 기기 안에만 저장됩니다.

**Q. 기기를 바꾸려면?**
A. 설정에서 **백업 내보내기** → 새 기기에서 **복원** 하세요.
